Overview

## Description

Excel VBAで使用可能な、配列クラスです。  
標準の配列と異なり、メソッドやプロパティにより操作ができます。  
メソッドなどはJavaやJavaScriptを踏襲して実装しています。  
使い方は[Demo](./#Demo)を確認してください。
